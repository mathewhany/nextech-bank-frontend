# How to Run Locally?
1. Clone 
```bash
git clone https://github.com/software-engineering-2023/Batates-M7mra
```
2. Open the cloned repo
```bash
cd Batates-M7mra
```
3. Install dependencies
```bash
npm install
```
4. Run the local server
```bash
npm run dev
```
5. Open [http://localhost:3000](http://localhost:3000) with your browser.
